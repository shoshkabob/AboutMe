SpeakSruly

SpeakSruly is a fun little project that my good friend and UX designer Chava Weisberg came up with and designed. We have a friend named Sruly who often posts enigmatic, eloquent Facebook statuses that not many can understand; so she thought, what if we could translate them for everyone? I built a Sinatra server that skips over designated basic English words to translate any more complicated English sentences. Since machine translation is a complicated science that involves many expensive APIs that I did not feel were a necessary investment for a fun little side project, the translation is rudimentary and often hilarious.
Github

HarmonyTime

HarmonyTime utilizes the Web Audio API to play notes and recognize pitch, and a notation language called VexFlow to draw music notes right onto the webpage using HTML Canvas. I used a Ruby Sinatra server and SQLite3 database on the back-end, and JQuery and Javascript for client-side scripting.
Github
Note: This project is currently under construction. Better servers and more features to come!
